# Harbour
Docker/Portainer management app for iOS.

## Features
- [x] Multiple servers & endpoints
- [x] Managing container state
- [x] Managing stacks
- [x] Widgets
- [x] Handoff
- [x] iPadOS / macOS Catalyst support
- [x] No telemetry :)
